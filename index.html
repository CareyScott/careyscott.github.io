Prontotyper (2022)

Recently, I undertook a project where I developed an Object Detection model. The purpose of this model was to recognise and predict hand drawn images of web elements. To accomplish this, I made use of a React.js frontend and REST API. This enabled me to serve the prediction model and generate the necessary code for the user to employ in real time. As the user interfaced with the model, it generated appropriate HTML code which the user could edit and use or implement in real time. In order to enhance the model's effectiveness, I used two Azure storage solutions. This involved making use of both Blobs and Cosmos DB with mongoose schema. Lastly, to refine the model's ability to accurately predict and recognise web elements, I made use of Azure cognitive services. By doing so, I was able to ensure that the model was highly accurate in its predictions and easy to use for the end user.
