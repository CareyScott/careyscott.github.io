Intro

Hello!

My Name is Scott.

I'm a 22 year old student living and studying in Dublin, Ireland. I have a huge passion for Computing, Music, Photography.

I study Creative Computing in the Dun Laoghaire Institute of Art, Design, and Technology. I have learned to combines both technical and design skills within this course, which has allowed me to experience programming, software engineering, networking, database design and development, computer system architectures with an emphasis on hands-on and practical experiences.

I’m currently working on my final capstone project which involves design and development of an object detection and prediction system, which automatically converts into fully functioning web pages – using React.js, REST APIs and an Azure DB backend.

A core element of the work that I have completed also includes development of client-side GUI and related functionality plus back-end DB and related API calls to link the two components together.

Over the course of my studies, I have utilized and become proficient with the following programming languages and libraries:

Leveraging the above technologies, I have built high levels of expertise in the design and build of both client and server-side applications and APIs which primarily run with the JavaScript programming language and its coexisting libraries. These also include any supporting network and TCP/IP related settings, configurations, ports, and system calls.

As part of my capstone project for college, I developed an Object Detection model which recognises and predicts hand drawn images of web elements in conjunction with a React.js frontend and REST API to serve the prediction model and generate appropriate code for the user to be able to use in real time. Utilising two Azure storage solutions (Blobs & Cosmos DB with mongoose schema) and Azure cognitive services.

I have previous experience in customer interactions, the maintaining of Virtual Reality hardware and software, and have excellent teamwork and collaboration skills.
